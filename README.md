# ipv4-to-int

## Install

```
npm install ipv4-to-int --save
```

## Usage

```javascript
const ipInt = require('ipv4-to-int');

// IP to Integer
ipInt('89.150.79.17').toInt(); // 1503022865

// Integer to IP
ipInt('1503022865').toIP(); // 89.150.79.17
```
